Explainable Diabetes Classification App

This project is an interactive machine learning application that predicts the likelihood of diabetes using the Pima Indians Diabetes Dataset. It also provides visualizations and explanations to help users understand how the model makes its predictions.

Project Structure

- app.py – The main application file. It includes the user interface, data preprocessing, model training, prediction logic, and data visualizations.
- requirements.txt – Lists all the Python libraries required to run the project.
- data/ – Contains the Pima Indians Diabetes Dataset used for training and testing the model.

How to Run the Project

1. Make sure Python is installed on your computer.
2. Open a terminal or command prompt in the project folder.
3. Install the required dependencies by running:

- cmd
pip install -r requirements.txt

4. Start the Streamlit application:

- cmd
streamlit run app.py


5. Once the app starts, it will automatically open in your default web browser. If it doesn't, copy and open the local URL displayed in the terminal.
